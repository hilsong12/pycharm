# Machine Learning Study Repository 📚

이 저장소는 머신러닝과 딥러닝을 학습하면서 작성한 코드들을 정리한 개인 프로젝트 공간입니다.  
각 Python 파일은 개별 주제에 대한 실습 코드이며, 데이터 처리 → 모델 학습 → 평가까지의 과정을 포함하고 있습니다.

---

## 🧠 포함된 학습 내용

| 파일 이름 | 주제 |
|----------|------|
| `ai_exam01_introduction_of_deeplearning.py` | 딥러닝 개요 및 기본 구조 |
| `ai_exam02_principles_of_deeplearning.py` | 딥러닝 학습 원리 |
| `ai_exam03_keras_xor.py` | XOR 문제 해결 모델 |
| `ai_exam04_heart_disease_classification.py` | 심장병 예측 모델 |
| `ai_exam05_titanic.py` | Titanic 생존 예측 |
| `ai_exam06_iris_classification.py` | Iris 데이터 분류 |
| `ai_exam07_classification_mnist.py` | MNIST 숫자 분류 |
| `ai_exam08_classification_fashion_mnist.py` | Fashion-MNIST 의류 데이터 분류 |
| `ai_exam09_1_image_convert.py` | 이미지 데이터셋 전처리 |
| `ai_exam09_2_cat_and_dog.py` | 고양이/강아지 이미지 분류 모델 |
